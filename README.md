## Spring Security
Spring Security是一个功能强大且高度可定制的身份验证和访问控制框架。且专注于为Java应用程序提供身份验证和授权的框架。

通过本项目可以迅速深入理解Spring Security底层的鉴权流程，以及更多的扩展鉴权方式，不仅可以提高和突破技术瓶颈，更能在项目开发中迅速集成权限框架。
